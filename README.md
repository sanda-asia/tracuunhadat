## Tenchnical stack: MEVN stack
- **Database**: MongoDB
- **Web Framwork**: ExpressJS
- **Frontend Framwork**: VueJS
- **Platform**: NodeJS


## Hướng dẫn chạy:

1. Cài đặt tất cả các dependencies cần thiết trong project  
	```
    npm i
    ```
2. Chạy server
	```
    npm start
    ``` 
    hoặc 
    ```
    node server.js
    ```
3.  Chạy client
	```
    npm run dev
    ```
